8.	Тестовое задание на знание контейнеризации и процессов CI/CD

      a.	Установить на сервер Git, либо воспользоваться общедоступным.  
      b.	Развернуть инсталяцию Jenkins.  
      c.	В Jenkins создать job которая будет загружить код из мастер ветки Git в локальное хранилище.  
      d.	Создать Job_2 которая удалит из проекта 1-2 файла.  
      e.	Создать Job_3 которая вернет проект в Git.  
      f.	Необходимо, что бы Job вызывались автоматически, а не руками.
